# 🏜️ The Desert Breakdown
A survival game where you wake up next to a broken-down car in the middle of the desert. Your goal: find the missing parts, repair the car, and escape before you starve or die of thirst.

## 🎮 Gameplay
- **Explore** the desert and find scattered car parts, food, and water
- **Survive** by managing your hunger and thirst
- **Repair** the car by installing all missing parts
- **Escape** through the tunnel once the car is fully repaired

## 🕹️ Controls
| Key | Action |
|-----|--------|
| `WASD` | Move |
| `Mouse` | Look around |
| `Left Shift` | Sprint |
| `Space` | Jump |
| `E` | Pick up item |
| `Hold F` | Install car part |
| `Tab` | Open/close inventory |
| `1-4` | Select hotbar slot (also consumes food/water) |
| `Right Click` | Use item in inventory |
| `ESC` | Pause menu |
| `R` | Respawn after death |

## 🚗 Car Parts to Find
- 4× Wheel
- Left Door
- Right Door
- Steering Wheel

## ⚙️ Features
- Inventory system with drag & drop (hotbar + backpack)
- Survival stats: HP, Food, Water
- Death & respawn system with checkpoints
- Save/load system
- Car repair with hold-to-install progress bar
- Cinematic ending cutscene
- Scrolling credits
